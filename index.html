🤖 Autonomous AGV Logistics Platform

ROS 2 + Nav2 + Full-Stack · desarrollado desde cero y evolucionado por fases

Sistema completo de logística AGV en entorno industrial simulado: desde la navegación autónoma básica hasta un dashboard de monitorización en tiempo real.

ROS 2 Jazzy Nav2 Python Gazebo Sim Safety Zones FastAPI WebSocket React
  • Navegación autónoma: Nav2 y local costmaps para evasión de obstáculos dinámicos.
  • Ejecución de misión: máquina de estados en Python (approach, pickup, loading, dropoff, unloading).
  • Supervisión de seguridad: zonas SAFE / SLOW / STOP / RESTRICTED con un nodo ROS 2 desacoplado.
  • Dashboard full-stack: ROS 2 → FastAPI → WebSocket → React, con posición, velocidad, misión y safety log en vivo.
